# 📈 Indian Stock Market Price Analysis

A data analysis project that explores historical stock price trends in the Indian stock market using Python. The project uses data from the National Stock Exchange (NSE) and Bombay Stock Exchange (BSE) to perform exploratory data analysis, visualize stock performance, and generate insights for investors and analysts.

## 📌 Project Overview

The Indian stock market is one of the world's fastest-growing financial markets. This project analyzes historical stock price data to identify trends, price movements, volatility, and market behavior.

The analysis includes:

- Historical stock price analysis
- Daily return calculation
- Moving averages
- Volatility analysis
- Trading volume analysis
- Data visualization
- Comparative stock performance

India's stock market is primarily driven by the NSE and BSE, which together host thousands of listed companies. :contentReference[oaicite:0]{index=0}

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/taufique97alam-arch/Indian-Stock-Market-Analysis.git
```

Navigate to the project directory:

```bash
cd Indian-Stock-Market-Analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Running the Project

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```text
stock_analysis.ipynb
```

Run all cells to reproduce the analysis.
